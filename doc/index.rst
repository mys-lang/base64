About
=====

Base64 encoding and decoding in the `Mys programming language`_.

Project: https://github.com/mys-lang/package-base64

Example
=======

.. code-block:: python

   from base64 import encode
   from base64 import decode

   def main():
       print(encode(b"foo"))
       print(decode("Zm9v"))
